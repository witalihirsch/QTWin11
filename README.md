# QTWin11
QTWin11 is a template to customize win32 apps using QT. 

## Features
- This template uses the system accent color
- Used Mica effect on win32 apps

## Installing
```pwsh
pip install PySide2
```
```pwsh
pip install win32mica
```
```pwsh
pip install darkdetect
```
## Requirements
- Windows 11
